## 典型问题示例

### 问题1：连续列表无过渡

**不好的写法**：

```markdown
### 功能特点

- 功能A
- 功能B
- 功能C

### 使用方法

- 步骤1
- 步骤2
- 步骤3
```

**改进写法**：

```markdown
### 功能特点

OpenClaw提供了三大核心功能。首先是功能A，它可以帮助用户快速完成任务。
其次是功能B，专门用于处理复杂场景。最后是功能C，提供了高级定制能力。

### 使用方法

了解了功能特点后，下面介绍具体的使用方法。整个过程分为三个步骤，
每个步骤都有详细的说明。

第一步是初始化配置。打开配置文件，设置必要的参数...

第二步是启动服务。执行启动命令后，系统会自动...

第三步是验证结果。通过查看日志，可以确认...
```

### 问题2：小标题过多导致碎片化

**不好的写法**：

```markdown
#### 优势1
- 说明

#### 优势2
- 说明

#### 优势3
- 说明
```

**改进写法**：

```markdown
#### 核心优势

OpenClaw具有三大核心优势。

在性能方面，OpenClaw采用了先进的架构设计，响应速度快，资源占用低。
即使处理大量数据，也能保持流畅运行。

在易用性方面，OpenClaw提供了直观的界面和简单的配置方式。新手用户
可以在几分钟内完成基本设置，无需复杂的学习过程。

在扩展性方面，OpenClaw支持丰富的插件系统。用户可以根据自己的需求，
安装各种Skills来扩展功能，满足不同场景的使用需求。
```
