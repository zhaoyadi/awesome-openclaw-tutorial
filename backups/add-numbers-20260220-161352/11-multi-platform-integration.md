## 为什么需要多平台集成？

OpenClaw支持接入多个IM平台，让你可以：

- 📱 **随时随地使用**：手机、电脑、平板都能访问
- 👥 **团队协作**：多人共享AI助手，提高团队效率
- 🔄 **跨平台同步**：不同平台间数据互通
- 🎯 **场景适配**：工作用飞书，生活用QQ，社交用Discord

## 通用配置流程

所有平台的配置流程基本相同，分为4个步骤：

### 步骤1：创建应用/机器人

1. **注册开发者账号**
   - 访问平台开发者中心
   - 注册账号（如果还没有）
   - 完成实名认证（部分平台需要）

2. **创建应用**
   - 进入应用管理
   - 创建新应用
   - 填写应用信息

3. **获取凭证**
   - App ID / Bot Token
   - App Secret / API Key
   - 其他必要凭证

### 步骤2：配置OpenClaw

1. **安装对应Skills**
   ```bash
   # 飞书
   openclaw plugins install @openclaw/feishu
   
   # 企业微信
   openclaw plugins install @openclaw/wework
   
   # 钉钉
   openclaw plugins install @openclaw/dingtalk
   ```

2. **配置凭证信息**
   ```bash
   # 方式1：使用配置向导
   openclaw onboard
   
   # 方式2：手动配置
   openclaw config set channels.feishu.appId "cli_xxx"
   openclaw config set channels.feishu.appSecret "xxx"
   ```

3. **启动服务**
   ```bash
   # 重启Gateway
   openclaw gateway restart
   ```

### 步骤3：测试验证

1. **发送测试消息**
   - 在IM平台中找到机器人
   - 发送"你好"测试
   - 等待回复

2. **验证回复**
   - 检查是否收到AI回复
   - 验证回复内容是否正确
   - 测试其他功能

3. **检查日志**
   ```bash
   # 查看Gateway日志
   tail -f ~/.openclaw/logs/gateway.log
   ```

### 步骤4：访问控制

1. **配置白名单**
   ```bash
   # 添加允许的用户
   openclaw config set channels.feishu.allowlist '["user_id_1","user_id_2"]'
   ```

2. **设置权限**
   - 配置管理员
   - 设置功能权限
   - 限制访问范围

3. **群组管理**
   - 添加机器人到群组
   - 配置群组权限
   - 设置群组白名单

## 常见问题

### Q1: 应该选择哪个平台？

**A**: 根据你的使用场景选择：
- **个人使用** → 飞书（功能最完整）
- **企业使用** → 企业微信（与微信互通）
- **办公自动化** → 钉钉（审批流程）
- **国外用户** → Telegram（全球覆盖）

### Q2: 可以同时配置多个平台吗？

**A**: 可以！OpenClaw支持同时接入多个平台，你可以：
- 工作用飞书
- 生活用QQ
- 社交用Discord

### Q3: 配置后无法收到消息怎么办？

**A**: 按以下步骤排查：
1. 检查Gateway是否运行：`openclaw gateway status`
2. 查看日志：`tail -f ~/.openclaw/logs/gateway.log`
3. 验证凭证是否正确
4. 检查网络连接
5. 查看对应平台的故障排查章节

### Q4: 如何配置多个Agent？

**A**: 查看对应平台的多Agent配置章节：
- [飞书多Agent配置](09-1-feishu-bot.md#多agent配置)
- [企业微信多Agent配置](09-2-wework-dingtalk.md#多agent配置)

### Q5: 配置成本是多少？
